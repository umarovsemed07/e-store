Overview
This repository contains a Power BI project (e-store.pbix) designed to analyze and visualize the sales performance of an e-commerce store. The dashboard provides actionable insights into revenue trends, product performance, and customer purchasing behaviors to support data-driven decision-making.

Key Features
Sales Overview: Comprehensive view of Total Revenue, Total Orders, and Profit Margins.
Trend Analysis: Month-over-month (MoM) and year-over-year (YoY) sales comparisons to identify seasonal trends.
Product Performance: Identification of top-selling and least-performing product categories.
Interactive Filtering: Dynamic slicers for filtering data by date, category, and region to allow for granular analysis.

Tools & Technologies Used
Power BI Desktop: For data modeling, visualization, and dashboard design.
Power Query: For data cleaning, transformation, and shaping before loading into the model.
DAX (Data Analysis Expressions): Used to create custom calculated columns and dynamic measures (e.g., Year-to-Date sales, Profit Margins).

Dataset
The data used in this project represents an e-commerce transaction log, including tables for Orders, Products, and Customers. (Note: Sensitive information has been anonymized/removed for public display).

How to Open This Project
Download and install Microsoft Power BI Desktop.
Clone or download this repository to your local machine.
Open the e-store.pbix file using Power BI Desktop.
Interact with the visuals using the provided slicers and filters.
